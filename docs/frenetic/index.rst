FRENETIC
==================
FRENETIC_ is a multi-physics code developed at Politecnico di Torino for the steady state and transient simulation of the core of liquid metal-cooled fast reactors.


.. _FRENETIC: https://bitbucket.org/gfnallo/frenetic/src/master/
